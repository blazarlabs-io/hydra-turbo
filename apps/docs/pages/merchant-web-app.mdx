# Merchant Web App

**Web-based merchant dashboard and payment management system**

The Merchant Web App provides a comprehensive dashboard for businesses to manage their Blazar Terminal payments, view transaction history, and configure payment settings.

## Overview

The Merchant Web App is a modern web application that enables merchants to:

- **Monitor Payments**: Real-time payment monitoring and notifications
- **Manage Terminals**: Configure and manage multiple Blazar Terminals
- **View Analytics**: Comprehensive payment analytics and reporting
- **Process Refunds**: Handle refunds and payment disputes
- **Configure Settings**: Customize payment preferences and terminal settings

## Technology Stack

- **Frontend**: React with TypeScript
- **Backend**: Node.js with Express
- **Database**: PostgreSQL with Redis caching
- **Authentication**: JWT with OAuth2 support
- **Real-time**: WebSocket connections to Blazar Terminals
- **Deployment**: Docker containers with Kubernetes orchestration
