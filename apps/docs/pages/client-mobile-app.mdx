# Client Mobile App

**Mobile wallet integration for seamless payment experiences**

The Client Mobile App enables users to make contactless payments through Blazar Terminals using their Cardano mobile wallets.

## Overview

The Client Mobile App provides a seamless mobile payment experience that allows users to:

- **Discover Terminals**: Automatically find nearby Blazar Terminals
- **Make Payments**: Process secure micropayments using Cardano
- **Manage Wallets**: Connect and manage multiple Cardano wallets
- **View History**: Track payment history and transaction details
- **Security Features**: Biometric authentication and secure key management

## Supported Platforms

- **iOS**: Native Swift/Objective-C implementation
- **Android**: Native Kotlin/Java implementation
- **React Native**: Cross-platform solution
- **Flutter**: Cross-platform solution

## Technology Stack

- **Mobile Framework**: React Native / Flutter
- **BLE Library**: react-native-ble-plx / flutter_blue
- **Crypto**: Cardano crypto libraries
- **Storage**: Secure keychain/keystore integration
- **Authentication**: Biometric authentication (Touch ID, Face ID, Fingerprint)
